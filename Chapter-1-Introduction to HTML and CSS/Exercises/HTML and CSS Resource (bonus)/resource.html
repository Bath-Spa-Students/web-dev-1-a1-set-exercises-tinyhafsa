HTML Tags

HTML stands for Hyper Text Markup Language. It describes the structure of a Web page. It's made up of elements that tell the browser how to display content.

Types of HTML tags

Container Tags are tags that require both opening and closing tags. these include heading tags and anchor tags.
Empty Tags are tags that do not require a closing tag.

Page Structure

1. Structural Elements

The <!DOCTYPE html> declaration defines that this document is an HTML5 document.
The <html lang="en"> element is the root element of an HTML page.
The <head> element contains meta information about the HTML page.
The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab).
The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc. 

                    
                        <!DOCTYPE html>
                        <html lang="en">
                        <head> 
                            <title>My First Web Page</title> 
                        </head> 
                        <body>
                            <h1>Hello, World!</h1> 
                            <p>This is my first web page.</p>
                        </body> 
                        </html>

Basic Tags

2. Paragraph

The <p> tag is used to add a paragraph to the page.

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>

3. Headings

HTML headings are titles or subtitles that you want to display on a webpage. HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading.

                
                    <h1>Heading Level 1</h1>
                    <h2>Heading Level 2</h2>
                    <h3>Heading Level 3</h3>
                    <h4>Heading Level 4</h4>
                    <h5>Heading Level 5</h5>
                    <h6>Heading Level 6</h6>

4. Line Break

The <hr> is most often displayed as a horizontal rule that is used to separate content (or define a change) in an HTML page.

                
                    This is a paragraph above the horizontal rule.
                    <hr>
                    This is a paragraph below the horizontal rule.

5. Links

The HTML tag defines a hyperlink. It has the following syntax: <a href="url">link text</a>. The most important attribute of the element is the href attribute, which indicates the link's destination. The link text is the part that will be visible to the reader.

                
                    <a href="https://www.example.com">Visit Example.com</a>

6. Div and Span

The <div> element is a block-level elemnt used to create a container that can hold other HTML elements. The <span> tag is an inline container used to mark up a part of a text, or a part of a document. Both tags can be styled with CSS or manipulated with JavaScript.

                
                    <div id="container">
                        <p>This is a paragraph inside a div.</p>
                        <span>This is a span inside the div.</span>
                    </div>

Lists

7. Unordered List

The <ul> tag defines an unordered (bulleted) list. The <li> elements represent list items within the list. 
                
                    <ul>
                        <li>Item 1</li>
                        <li>Item 2</li>
                        <li>Item 3</li>
                    </ul>

8. Ordered List

The <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical.The <li> tag is used to define each list item.

                
                    <ol>
                        <li>First item</li>
                        <li>Second item</li>
                        <li>Third item</li>
                    </ol>

Media

9. Images

The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image. The alt specifies an alternate text for the image, if the image for some reason cannot be displayed

                
                    <img src="example.jpg" alt="Example Image">

10. Tables

The <table> tag defines an HTML table. It is made up of one <table> element and one or more <tr>, <th>, and <td> elements.
The <tr> element defines a table row, the <th> element defines a table header, and the <td> element defines a table cell.

                
                    <table>
                        <tr>
                            <th>Header 1</th>
                            <th>Header 2</th>
                        </tr>
                        <tr>
                            <td>Data 1</td>
                            <td>Data 2</td>
                        </tr>
                        <tr>
                            <td>Data 3</td>
                            <td>Data 4</td>
                        </tr>
                    </table>

CSS Properties

CSS stands for Cascading Style Sheets. It is the language we use to style an HTML document. It describes how HTML elements should be displayed.

Adding CSS to a Webpage

There are three ways to add CSS to a webpage:

  • External CSS: it is contained in an external file and is can be used to style an entire website.
  • Internal CSS: it is contained in the head tag of the page and is used to style one page of a website.
  • Inline CSS: it is contained in an element and is used to style that specific element.

CSS Syntax

A CSS rule consists of a selector and a declaration block. The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon. Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

                    
                        p {
                            color: red;
                            text-align: center;
                        }

In the above code p is the selector. color and text-align are properties, with values of red and center.

1. Color

The color of the text can be changed using the color property. It is given a value of a predefined color name, HEX code or RGB value. 

                
                    p {
                        color: blue;
                    }

2. Text Decoration

The text-decoration property is used to add decoration to a text by adding a line to it, and then changing its color, style and thickness.

                
                    p {
                        text-decoration: underline red double 5px;
                    }

3. Text Alignment

The text-align property is used to set the horizontal alignment of a text. A text can be left or right aligned, centered, or justified.

                
                    h1 {
                        text-align: center;
                    }

4. Background Color

The background-color property specifies the background color of an element.

                
                    body {
                        background-color: lightblue;
                    }

5. Background Image

The background-image property specifies an image to use as the background of an element. 

                
                    body {
                        background-image: url('example-image.jpg');
                    }

6. Background Repeat

By default, the image is repeated vertically and horizontally so it covers the entire element. The value can be changed to set the image to repeat only horizontally repeat-x, or vertically repeat-y or not at all no-repeat.

                
                    body {
                        background-image: url("img_tree.png");
                        background-repeat: no-repeat;
                    }

7. Border

The CSS border property specifies the style, width, and color of an element's border.
The border-style property specifies what kind of border to be displayed, like dotted, dashed or solid.
The border-width property specifies the width of the four borders.
The border-color property is used to set the color of the four borders.
The border-radius property allows rounded corners to be set. 
                
                    p {
                        border-style: solid;
                        border-width: 2px;
                        border-color: red;
                        border-radius: 4px;
                    }

8. Text Shadow

The text-shadow property applies shadow to text. Values for the horizontal and vertical shadows can be specified, and color and a blur effect can be added to the text.

                
                    p {
                        text-shadow: 2px 2px red 1px;
                    }

9. Box Shadow

The box-shadow property is used to apply one or more shadows to an element. Values for horizontal and vertical shadows, colors and blur effects can be added.

                
                    div {
                        box-shadow: 10px 10px 5px lightblue;
                    }

10. Font

The font-family property allows the name of the font to be added as a value.
The font-size property sets the size of the font.
The font-size property style the text to be italic or oblique 

                
                    p {
                        font-family: Arial, sans-serif;
                        font-size: 18px;
                        font-style: italic;
                    }
